# 💰 AssetFlow — 자산 흐름 시뮬레이터

초기 자산, 월 수입/소비, 투자 수익률 등을 입력하면 **연도별 자산 변화**와 **목표 자산 달성 시점**을 시뮬레이션하는 개인 재무 분석 도구입니다.

![Python](https://img.shields.io/badge/Python-3.10%2B-blue?logo=python)
![Streamlit](https://img.shields.io/badge/Streamlit-1.32%2B-red?logo=streamlit)
![Plotly](https://img.shields.io/badge/Plotly-5.20%2B-3F4F75?logo=plotly)

---

## ✨ 주요 기능

- **투자 수익률 범위 시뮬레이션** — 최소/중간/최대 3개 시나리오를 동시에 비교
- **목표 자산 달성 시점** — SWR(안전 인출률) 기반 목표 자산선과 달성 연도 자동 계산
- **물가 상승률 / 연봉 상승률** — 매년 소비와 수입에 복리 반영
- **은퇴 시나리오** (선택) — 은퇴 시점, 은퇴 후 수입·수익률·세금(gross-up)을 분리 계산
- **연도별 상세 테이블** — 월 수입, 월 소비, 재투자액, 연수익, 누적 자산을 연도별로 확인

## 📖 사용 방법

### 기본 설정 탭 (💼)

| 항목 | 설명 |
|---|---|
| 초기 투자 자산 | 시뮬레이션 시작 시점의 보유 자산 (만원) |
| 월 수입 | 세후 월 수령 금액 (만원) |
| 월 소비 | 매월 지출 금액 (만원) |
| 투자 수익률 범위 | 연 수익률 최솟값 ~ 최댓값 (%) |
| 시뮬레이션 기간 | 몇 년 후까지 계산할지 (년) |
| 물가 상승률 | 매년 월 소비가 증가하는 비율 (%) |
| 연봉 상승률 | 매년 월 수입이 증가하는 비율 (%) |
| 안전 인출률 (SWR) | 자산 고갈 없이 매년 인출 가능한 비율. 기본값 4% (Trinity Study 기준) |

### 은퇴 시나리오 탭 (🏖️)

오른쪽 사이드바 상단의 **시나리오 켜기** 토글을 활성화하면 은퇴 조건을 추가로 설정할 수 있습니다.

| 항목 | 설명 |
|---|---|
| 은퇴 시점 | N년차부터 아래 조건 적용 |
| 은퇴 후 월 수입 | 연금·임대·파트타임 등 투자 외 수입 (만원) |
| 인출 세금률 | 부족분 인출 시 적용되는 세율. 해외주식 양도세 기본 22% |
| 은퇴 후 투자 수익률 | 은퇴 후 안정적인 포트폴리오 수익률 범위 (예: 3.5% ~ 7.5%) |

> **인출 세금 계산 방식 (gross-up)**  
> 월 부족분 200만원, 세율 22% → 실제 매도액 = 200 ÷ (1 − 0.22) ≈ 256만원

## 📐 계산 공식

### 자산 성장

```
자산(n+1) = 자산(n) × (1 + 수익률) + 연 저축액
연 저축액  = (월 수입 − 해당 연도 월 소비) × 12
월 소비    = 초기 월 소비 × (1 + 물가상승률)^n
월 수입    = 초기 월 수입 × (1 + 연봉상승률)^n  ← 은퇴 전까지만 적용
```

### 목표 자산 (SWR 기반)

```
목표 자산 = 해당 연도 연간 소비 ÷ SWR
```

이 자산에서 매년 SWR%씩 인출하면 이론적으로 자산이 고갈되지 않습니다.
