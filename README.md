# 💰 AssetFlow — 자산 흐름 시뮬레이터

초기 자산, 월 수입/소비, 투자 수익률 등을 입력하면 **연도별 자산 변화**, **목표 자산 달성 시점**, **은퇴 후 자산 고갈 위험**까지 시뮬레이션하는 개인 재무 분석 도구입니다.

![Python](https://img.shields.io/badge/Python-3.10%2B-blue?logo=python)
![Streamlit](https://img.shields.io/badge/Streamlit-1.32%2B-red?logo=streamlit)
![Plotly](https://img.shields.io/badge/Plotly-5.20%2B-3F4F75?logo=plotly)

---

## ✨ 주요 기능

- **3-시나리오 비교** — 최소/중간/최대 수익률을 동시에 시뮬레이션 (밴드 + 곡선)
- **목표 자산 (SWR 기반)** — 인플레이션 반영 목표 자산선, 달성 연도 자동 표기
- **물가 / 연봉 상승률** — 매년 소비·수입에 복리 반영
- **은퇴 시나리오** — 은퇴 시점, 이후 기간, 은퇴 후 수입·수익률·세금(gross-up) 분리 계산
  - 자산 시나리오 mid 곡선이 은퇴 시점에 도달한 자산값을 시작점으로 한 점에서 분기
- **자산 감소 단계별 경고** — 🟡 자산 반토막(-50%) / 🟠 1/5 남음(-80%) / 💀 고갈, 곡선 위 마커로 표시
- **Monte Carlo 확률 분석** — 임계값 누적 도달 확률을 두 신뢰구간(2σ vs 1.2σ)으로 비교
- **연도별 상세 표** — 자산 시나리오 / 은퇴 시나리오 별도 expander로 분리

## 📖 사용 방법

### 💼 자산 시나리오 탭

| 항목 | 설명 |
|---|---|
| 초기 투자 자산 | 시뮬레이션 시작 시점의 보유 자산 (만원) |
| 월 수입 | 세후 월 수령 금액 (만원) |
| 월 소비 | 매월 지출 금액 (만원) |
| 투자 수익률 범위 | 연 수익률 최솟값 ~ 최댓값 (%) |
| 시뮬레이션 기간 | 몇 년 후까지 계산할지 (은퇴 OFF일 때 사용) |
| 물가 상승률 | 매년 월 소비가 증가하는 비율 (%) |
| 연봉 상승률 | 매년 월 수입이 증가하는 비율 (%) — 은퇴 후엔 미적용 |
| 안전 인출률 (SWR) | 자산 고갈 없이 매년 인출 가능한 비율. 기본값 4% (Trinity Study 기준) |

### 🏖️ 은퇴 시나리오 탭

사이드바 상단의 **시나리오 켜기** 토글을 활성화하면 은퇴 조건이 추가됩니다.

| 항목 | 설명 |
|---|---|
| 은퇴 시점 (N년차) | 은퇴 시작 연도. 자산 시나리오 그래프의 중간(평균) 곡선을 보고 결정 |
| 은퇴 이후 기간 (년) | 은퇴 이후 자산 흐름을 추적할 기간 |
| 은퇴 후 월 수입 | 연금·임대·파트타임 등 투자 외 수입 (만원) |
| 인출 세금률 | 부족분 인출 시 적용되는 세율. 해외주식 양도세 기본 22% |
| 은퇴 후 투자 수익률 | 은퇴 후 안정적 포트폴리오 수익률 범위 (예: 3.5% ~ 7.5%) |

> **은퇴 시작 자산 결정 방식**  
> 자산 시나리오의 mid(중간) 곡선이 은퇴 시점에 도달한 자산값 **한 점**을 잡고,
> 거기서 은퇴 후 min/mid/max 수익률에 따라 분기합니다. "평균적인 자산 흐름을
> 보고 은퇴 결정" 하는 현실적 의사결정 흐름을 반영.

> **인출 세금 계산 방식 (gross-up)**  
> 월 부족분 200만원, 세율 22% → 실제 매도액 = 200 ÷ (1 − 0.22) ≈ 256만원

### 📊 확률 분석 (Monte Carlo)

은퇴 토글이 켜진 경우, **자산 감소 단계별 누적 도달 확률**을 Monte Carlo 시뮬레이션(10,000회)으로 계산해 표·곡선 차트로 표시합니다. 정규분포 가정 하에 두 신뢰구간을 비교:

| 가정 | σ | 의미 |
|---|---|---|
| **2σ 가정** | (max−min)/4 | 입력 범위를 95.4% 신뢰구간으로 해석 (표준) |
| **1.2σ 가정** | (max−min)/2.4 | 입력 범위를 ~77% 신뢰구간으로 해석 (변동성 큰 스트레스 시나리오) |

> ⚠️ 실제 시장 수익률은 fat-tail이라 정규분포 가정은 극단 위험을 과소추정할 수 있습니다. **참고용 추정치**로 활용하세요.

## 📐 계산 공식

### 자산 성장 (은퇴 전)

```
자산(n+1) = 자산(n) × (1 + 수익률) + 연 저축액
연 저축액  = (월 수입 − 해당 연도 월 소비) × 12
월 소비    = 초기 월 소비 × (1 + 물가상승률)^n
월 수입    = 초기 월 수입 × (1 + 연봉상승률)^n
```

### 은퇴 후 자산 변화

```
부족분(월) = 월 소비 − 은퇴 후 월 수입
매도액(월) = 부족분 ÷ (1 − 세율)            ← gross-up
자산(n+1)  = 자산(n) × (1 + 은퇴 수익률) − 매도액 × 12
```

### 목표 자산 (SWR 기반)

```
목표 자산 = 해당 연도 연간 소비 ÷ SWR
```

이 자산에서 매년 SWR%씩 인출하면 이론적으로 자산이 고갈되지 않습니다.

### 자산 감소 임계값 (은퇴 시나리오)

은퇴 시작 자산 대비 명목 기준:
- 🟡 자산 반토막 (-50%): 시작 자산의 50%
- 🟠 1/5 남음 (-80%): 시작 자산의 20%
- 💀 자산 고갈: 자산 ≤ 0

각 시나리오 곡선이 임계값을 처음 통과하는 시점에 마커 표시.
